Contract Rules

Ghosts cannot leave the investigation area.

Ghosts spawn in a random favorite room.

RAG REASONING PROTOCOL

When identifying ghosts:

Eliminate ghosts that do not match confirmed evidence.

Eliminate ghosts requiring evidence confirmed absent (default difficulty).

Apply guaranteed evidence rules for higher difficulties.

Apply Mimic rule if Ghost Orb present with UV + Spirit Box + Freezing.

Compare hunt threshold against current sanity.

Compare speed behavior.

Compare unique traits.

If one ghost remains → Identify it.

If multiple remain → Provide next best test.
